Dúvidas

Como foi realizada a integração com assistentes inteligentes, como Alexa e Google Home?

Nossa solução opera como um assistente inteligente independente. Para integrar com a Alexa, expusemos nossa API na nuvem e desenvolvemos uma skill personalizada da Alexa. Esta skill faz interface com nosso assistente, permitindo que ele demonstre todas as suas capacidades em um ambiente SmartHome e pode ser conferido atráves desse link: 

É fácil integrar essa solução as tecnologias já existentes da GoodWe?

Sim! Nossa solução utiliza comandos de API extraídos da versão Demo em https://www.semsportal.com. Para funcionar efetivamente, ela requer apenas a entrada de parâmetros já associados aos dispositivos GoodWe, como 'powerstationid' e 'devicesn'.

‍

 Como é feito a  automação residencial?

Criamos um sistema de IA que transforma dados brutos dos inversores GoodWe em insights acionáveis de otimização energética, automatizando decisões que tradicionalmente exigem intervenção manual especializada.

‍

Quais medidas são tomadas para evitar que o modelo de linguagem forneça respostas imprecisas?

Implementamos uma arquitetura de precisão onde o LLM funciona como interface inteligente, delegando cálculos sensíveis para funções Python especializadas. Isso garante que dados críticos sejam processados com confiabilidade, enquanto mantemos a naturalidade da interação. Todas as respostas incluem rastreabilidade para auditoria e validação.
